# 🌍 Desarrollo de un Sistema Automatizado para el Cálculo del Índice de Aridez  

## 📖 Descripción del Proyecto  
Este proyecto de tesis, desarrollado por **Norberto Arredondo Vizcarra** de la **Facultad de Informática de Culiacán**, tiene como objetivo implementar un sistema automatizado para calcular el **Índice de Aridez** en la **Cuenca III Pacífico Norte de México**.  

El índice de aridez es un indicador clave para entender las condiciones climáticas y su impacto en los ecosistemas, los recursos hídricos y la agricultura. Este sistema busca proporcionar herramientas precisas y accesibles para su cálculo.  

---

## 🔍 Características del Proyecto  
- **Lenguaje de Programación:** Python.  
- **Herramientas GIS:** Uso de aplicaciones como ArcMap y librerías especializadas como ArcPy para la manipulación y análisis geoespacial.  
- **Fuentes de Datos:** Información obtenida de fuentes gubernamentales de México, garantizando precisión y relevancia en el análisis.  

---

## 🎯 Objetivo  
Proveer un sistema eficiente y reproducible que automatice el cálculo del **Índice de Aridez**, facilitando su uso en:  
- Estudios climáticos.  
- Gestión ambiental.  
- Análisis hidrológicos.

Este sistema está diseñado para ser una herramienta accesible y práctica para:  
- **Investigadores:** Facilitando análisis complejos de datos climáticos.
