Este proyecto de tesis, desarrollado por Norberto Arredondo Vizcarra de la Facultad de Informática de Culiacán, tiene como objetivo implementar un sistema automatizado para calcular el Índice de Aridez en la Cuenca III Pacífico Norte de México.

Características del Proyecto
Lenguaje de Programación: Python.
Herramientas GIS: Uso de aplicaciones como ArcMap y librerías especializadas como ArcPy para la manipulación y análisis geoespacial.
Fuentes de Datos: Información obtenida de fuentes gubernamentales de México, garantizando precisión y relevancia en el análisis.

Objetivo
Proveer un sistema eficiente y reproducible que automatice el cálculo del Índice de Aridez, un indicador clave en estudios climáticos, hidrológicos y de gestión ambiental.

Tecnologías y Herramientas
ArcMap: Análisis y representación geográfica.
ArcPy: Automatización de procesos geoespaciales.
Python: Procesamiento de datos y desarrollo del sistema automatizado.

Este sistema busca ser una herramienta accesible para investigadores, académicos y tomadores de decisiones interesados en el monitoreo y análisis del impacto del clima en la región del Pacífico Norte.
